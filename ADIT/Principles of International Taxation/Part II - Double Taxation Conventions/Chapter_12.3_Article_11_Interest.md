## 1. INTRODUCTION: THE ROLE OF ARTICLE 11

### 1.1 Purpose and Function

Article 11 allocates taxing rights over **interest income** between the source state (where the payer is resident or where the debt is connected to a PE) and the residence state (where the recipient is resident). Like Article 10 (dividends), Article 11 uses a **"shared allocation"** approach with **limited source taxation**.

### 1.2 Interest as Passive Income

Interest, like dividends and royalties, is typically classified as **passive income**:
- Investment income without active participation
- Return on debt capital (vs equity capital for dividends)
- Often subject to WHT at source

### 1.3 Policy Rationale

```
SHARED ALLOCATION FOR INTEREST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SPLIT JURISDICTION APPROACH:
┌─────────────────────────────────────────────────────────────────┐
│ Both source and residence states have taxing rights:           │
│                                                                 │
│ • SOURCE STATE: May tax but limited to 10% (OECD)              │
│ • RESIDENCE STATE: May tax without limitation                  │
│ • RESIDENCE STATE: Must provide DTR (credit or exemption)      │
│                                                                 │
│ RATIONALE FOR SOURCE TAXATION:                                  │
│ → Debt-financed investment benefits source state economy       │
│ → Borrower uses source state infrastructure/legal system       │
│ → Some connection to where funds are used                      │
│                                                                 │
│ RATIONALE FOR LIMITATION:                                       │
│ → Avoid excessive double taxation                               │
│ → Encourage cross-border lending                                │
│ → Capital mobility concerns                                     │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 2. ARTICLE 11 TEXT AND STRUCTURE (OECD MTC 2017)

### 2.1 Full Text

**Article 11(1):**
> "Interest arising in a Contracting State and paid to a resident of the other Contracting State may be taxed in that other State."

**Article 11(2):**
> "However, such interest may also be taxed in the Contracting State in which it arises and according to the laws of that State, but if the beneficial owner of the interest is a resident of the other Contracting State, the tax so charged shall not exceed 10 per cent of the gross amount of the interest. The competent authorities of the Contracting States shall by mutual agreement settle the mode of application of this limitation."

## 3. ARTICLE 11(1) AND (2): TAXING RIGHTS ALLOCATION

### 3.1 Article 11(1): Residence State Right

```
RESIDENCE STATE TAXATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"Interest arising in a Contracting State and paid to a resident
of the other Contracting State may be taxed in that other State."

KEY ELEMENTS:
┌─────────────────────────────────────────────────────────────────┐
│ 1. "ARISING IN" a Contracting State                             │
│    → Source state identified                                    │
│    → See Article 11(5) for source rules                        │
│                                                                 │
│ 2. "PAID TO" a resident of other CS                             │
│    → Recipient must be resident of other CS                    │
│    → Third state residents not covered                          │
│                                                                 │
│ 3. "MAY BE TAXED" in that other State                           │
│    → Residence state has RIGHT but no obligation               │
│    → Unlimited taxing right (no cap)                           │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
SOURCE STATE TAXATION - 10% CAP
```

### 3.3 Beneficial Owner Requirement

Same concept as Article 10 (dividends):

```
BENEFICIAL OWNER OF INTEREST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

KEY DISTINCTION (Commentary):
┌─────────────────────────────────────────────────────────────────┐
│ Article 11 refers to beneficial owner of INTEREST               │
│ (NOT owner of the debt-claim)                                   │
│                                                                 │
│ This distinction is important where:                            │
│ → Debt-claim is owned by nominee                               │
│ → Interest collected by agent                                   │
│ → Conduit arrangement for interest                              │
└─────────────────────────────────────────────────────────────────┘

BENEFICIAL OWNER TEST:
┌─────────────────────────────────────────────────────────────────┐
│ • Has unconstrained right to use and enjoy the interest        │
│ • Not acting as agent, nominee, or conduit                     │
│ • Not under obligation to pass on to another person            │
│ • Not a narrow technical/legal ownership test                   │
│                                                                 │
│ CONDUIT = NOT beneficial owner                                  │
│ → Full domestic WHT applies (no 10% limitation)                │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. ARTICLE 11(3): DEFINITION OF "INTEREST"

### 4.1 Treaty Definition

```
DEFINITION OF "INTEREST" - ARTICLE 11(3)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

INCLUDED:
┌─────────────────────────────────────────────────────────────────┐
│ "Income from DEBT-CLAIMS of every kind"                         │
│                                                                 │
│ WHETHER OR NOT:                                                 │
│ • Secured by mortgage                                           │
│ • Carrying a right to participate in debtor's profits          │
│                                                                 │
│ IN PARTICULAR:                                                  │
│ • Income from government securities                             │
│ • Income from bonds or debentures                               │
│ • Premiums attaching to such securities                        │
│ • Prizes attaching to bonds                                     │
└─────────────────────────────────────────────────────────────────┘

EXCLUDED:
┌─────────────────────────────────────────────────────────────────┐
│ "Penalty charges for late payment shall NOT be regarded as     │
│ interest for the purpose of this Article"                       │
│                                                                 │
│ ALSO TYPICALLY NOT INTEREST:                                    │
│ • Capital gains on debt instruments (Article 13)               │
│ • Dividends (Article 10)                                        │
│ • Annuities (may be Article 18 or 21)                          │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 4.2 Key Distinctions

| Payment Type | Article | Notes |
|--------------|---------|-------|
| Interest on loan | Article 11 | Standard debt-claim |
| Interest on convertible bond | Article 11 | Until conversion |
| Premium on redemption | Article 11 | If predetermined |

## 5. ARTICLE 11(4): PE EXCEPTION

### 5.1 Effectively Connected Debt-Claims

```
PE EXCEPTION - ARTICLE 11(4)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RULE:
┌─────────────────────────────────────────────────────────────────┐
│ If interest is paid in respect of a debt-claim EFFECTIVELY     │
│ CONNECTED with a PE in the other state:                         │
│                                                                 │
│ → Article 11(1) and (2) do NOT apply                           │
│ → Article 7 (Business Profits) applies instead                 │
│ → Interest forms part of PE's business profits                 │
│ → No WHT limitation; PE taxation rules apply                   │
└─────────────────────────────────────────────────────────────────┘

"EFFECTIVELY CONNECTED" - same test as Article 10(4):
┌─────────────────────────────────────────────────────────────────┐
│ • More than merely recording debt in PE's books                │
│ • Genuine business connection required                          │
│ • Debt held for PE's business purposes                         │
│ • PE manages/administers the lending activity                  │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 6. ARTICLE 11(5): SOURCE OF INTEREST

### 6.1 Determination Rules

**EXAM RELEVANT**: Source determination affects which state can tax:

```
SOURCE OF INTEREST - ARTICLE 11(5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GENERAL RULE:
┌─────────────────────────────────────────────────────────────────┐
│ Interest arises in state where PAYER IS RESIDENT               │
│                                                                 │
│ Example:                                                        │
│ Company A (resident Country X) pays interest to Lender B       │
│ (resident Country Y)                                            │
│ → Interest "arises" in Country X                               │
│ → Country X is source state                                     │
└─────────────────────────────────────────────────────────────────┘

PE EXCEPTION:
┌─────────────────────────────────────────────────────────────────┐
│ Where payer has PE and:                                         │
│ • Indebtedness was incurred in CONNECTION with PE, AND        │
│ • Interest is BORNE BY the PE                                   │
│                                                                 │
│ → Interest arises in state where PE is situated                │
│ → NOT where payer is resident                                   │
│                                                                 │
│ Example:                                                        │
│ Company A (resident Country X) has PE in Country Z             │
│ PE borrows from Lender B (resident Country Y)                  │
│ Interest is borne by the PE                                     │
│ → Interest "arises" in Country Z (not Country X)               │
│ → Need Country Z-Country Y treaty (if any)                     │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 7. ARTICLE 11(6): ARM'S LENGTH PROVISION - EXAM CRITICAL

### 7.1 The Special Relationship Rule

**EXAM CRITICAL** (June 2022 Q6 extensively tested this provision):

```
ARTICLE 11(6) - ARM'S LENGTH FOR INTEREST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TEXT (paraphrased):
┌─────────────────────────────────────────────────────────────────┐
│ Where, by reason of a SPECIAL RELATIONSHIP between payer and   │
│ beneficial owner (or both and another person), the amount of   │
│ interest EXCEEDS the arm's length amount:                       │
│                                                                 │
│ → Article 11 applies ONLY to the arm's length amount           │
│ → EXCESS remains taxable under domestic law of each state      │
│ → Due regard to other Convention provisions                     │
└─────────────────────────────────────────────────────────────────┘

EFFECT:
┌─────────────────────────────────────────────────────────────────┐
│ ARM'S LENGTH INTEREST:                                          │
│ → Protected by Article 11                                       │
│ → 10% WHT cap applies                                           │
│                                                                 │
│ EXCESS INTEREST:                                                │
│ → NOT protected by Article 11                                   │
│ → Source state can apply full domestic WHT rates               │
│ → May be recharacterized as dividend (Article 10)              │
│ → "Secondary adjustment" concept (Bullen, 2011)                │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 7.3 Relationship Between Article 11(6) and Article 9

**EXAM CRITICAL** (June 2022 Q6 specific issue):

```
ARTICLE 11(6) vs ARTICLE 9 INTERACTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

THREE VIEWS (Kofler and Wittendorff, 2022):
┌─────────────────────────────────────────────────────────────────┐
│ VIEW 1: COMPLEMENTARY APPROACH (preferred in June 2022 exam)   │
│ • Article 9 and Article 11(6) have DIFFERENT FUNCTIONS         │
│ • Article 9: Enables adjustment to PROFITS of payer            │
│ • Article 11(6): Determines TREATY PROTECTION for interest     │
│ • Both can apply simultaneously                                 │
├─────────────────────────────────────────────────────────────────┤
│ VIEW 2: Article 11 takes precedence over Article 9              │
│ • Lex specialis - Article 11 is specific to interest           │
├─────────────────────────────────────────────────────────────────┤
│ VIEW 3: Article 9 does not apply to interest (or royalties)    │
│ • Articles 11 and 12 are self-contained                        │
└─────────────────────────────────────────────────────────────────┘

June 2022 EXAM APPROACH (Complementary):
┌─────────────────────────────────────────────────────────────────┐
│ 1. Article 11(6) determines whether EXCESS portion gets        │
│    Article 11 protection (WHT limitation)                       │
│                                                                 │
│ 2. Article 9 (with TPG Chapter X) determines:                   │
│    • Whether loan should be recharacterized as equity          │
│    • What arm's length interest rate should be                 │
│                                                                 │
│ 3. Both provisions work together but address different issues  │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 7.4 Article 11(6) Cannot Recharacterize the Loan

```
LIMITATION OF ARTICLE 11(6)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Commentary on Article 11(6), [35]:
┌─────────────────────────────────────────────────────────────────┐
│ Article 11(6) concerns itself ONLY with interest payments      │
│                                                                 │
│ → CANNOT recharacterize the NATURE of the loan                 │
│ → Cannot deem loan to be equity under Article 11(6) alone     │
│ → Can only exclude EXCESSIVE INTEREST from protection          │
│                                                                 │
│ For RECHARACTERIZATION of loan as equity:                       │
│ → Need domestic law provisions                                  │
│ → Article 9 and TPG Chapter X provide guidance                 │
│ → Some states amend Article 11(6) wording                      │
└─────────────────────────────────────────────────────────────────┘

DEDUCTION DENIAL:
┌─────────────────────────────────────────────────────────────────┐
│ Commentary [35] notes source state MAY have domestic rules     │
│ denying deduction for excessive interest portion               │
│                                                                 │
│ This is SEPARATE from Article 11(6) treaty qualification       │
└─────────────────────────────────────────────────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 8. UN MODEL DIFFERENCES

### 8.1 Key Differences Summary

```
OECD vs UN MODEL - ARTICLE 11 DIFFERENCES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────────────────────────────────────────────────────────────────┐
│               │ OECD MODEL          │ UN MODEL                  │
├───────────────┼─────────────────────┼───────────────────────────┤
│ WHT RATE      │ Fixed: 10%          │ Left to NEGOTIATION       │
│               │                     │ (no specified rate)       │
├───────────────┼─────────────────────┼───────────────────────────┤
│ POLICY        │ Lower source        │ Protects source state     │
│ ORIENTATION   │ taxation preferred  │ revenue for developing    │
│               │                     │ countries                 │
├───────────────┼─────────────────────┼───────────────────────────┤
│ "SUCH         │ Present (connects   │ Present (same as OECD)    │
│ INTEREST"     │ Art. 11(2) to       │                           │
│               │ Art. 11(1))         │                           │
└─────────────────────────────────────────────────────────────────┘

UN MODEL RATIONALE:
→ Developing countries are typically net borrowers
→ Interest flows OUT to developed country lenders
→ Leaving rates to negotiation allows higher source taxation
→ Protects developing country tax base
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 8.2 Practical Implications

Many actual DTAs use rates different from OECD 10%:
- Some reduce to 0% (especially UK treaties)
- Some exempt financial institution interest
- Developing countries may negotiate higher rates (15%+)

### 10.2 Treaty vs Domestic Law

| Issue | Article 11 (Treaty) | Domestic Law |
|-------|---------------------|--------------|
| **WHT rate** | Caps at 10% (or treaty rate) | May be higher absent treaty |

### 13.2 Article 11(6) Quick Reference

```
ARTICLE 11(6) SUMMARY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
